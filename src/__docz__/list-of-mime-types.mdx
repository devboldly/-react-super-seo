# List of MIME Types

When not provided, the following [MIME types](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types) are inferred from media extensions. 

This list includes many non-standard but widely accepted MIME types, such as `x-icon` for `.ico` files.
