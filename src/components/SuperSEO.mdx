<h1>SuperSEO</h1>

## Description

The React SEO component that handles metadata so you don't have to. Supports [Google](https://support.google.com/webmasters/answer/79812?hl=en), [OpenGraph](https://ogp.me/), [Twitter](https://developer.twitter.com/en/docs/tweets/optimize-with-cards/overview/abouts-cards), and beyond. 🚀

Supports the following:

- `title` and `description`
- [OpenGraph](https://ogp.me/) and [Twitter Cards](https://developer.twitter.com/en/docs/tweets/optimize-with-cards/overview/abouts-cards)
- [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) `lang` attribute in `html` element
- Google site verification
- Robots and [googlebot](https://support.google.com/webmasters/answer/79812?hl=en) configuration
- [Google-specific](https://support.google.com/webmasters/answer/79812?hl=en) meta tags
- Charset, viewport, and contentType
- [Rating](https://support.google.com/webmasters/answer/79812?hl=en) (for adult content)
- **Everything else!** Just wrap custom meta tags with `SuperSEO`

Uses sensible fallbacks where possible and features automatic [MIME type inference](/list-of-mime-types) for media URLs.

Includes full TypeScript type definitions and rich documentation, inside and out. 🙌

## Usage

```jsx
import { SuperSEO } from "react-super-seo";
```

Provide as little as a `title` and `description`, and go from there.

<Playground>

  <SuperSEO

## Custom Meta Tags

There are many more meta tags not covered here, such as the once popular, now obsolete `keywords` meta tag (now ignored by search engines), and others which you may be interested in but are outside the scope of this project.

If you need something not covered above, you can always provide them as children of `SuperSEO`.

Anything you wrap in the `SuperSEO` component will be rendered to the `head` element for you. Sky's the limit. 🚀

```jsx
<SuperSEO
  title="My Page"
  description="An awesome page that does a lot of cool stuff"
>
  <meta name="custom-meta" content="my-value" />
  <meta property="custom-meta-2" content="my-other-value" />
</SuperSEO>
```

> Add your custom meta tags inside the SuperSEO component.

## Examples

### Fallbacks

The following shows how fallbacks are used for OpenGraph:

```jsx
<SuperSEO
  title="SEO | React Super SEO"
  description="React SEO component with OpenGraph and Twitter Cards support."
  lang="en"
/>
```

This will be rendered as:

```html
<html lang="en" data-react-helmet="lang">
  <head>
    <title>SEO | React Super SEO</title>
    <meta
      name="description"
      content="React SEO component with OpenGraph and Twitter Cards support."
      data-react-helmet="true"
    />
    <meta
      property="og:title"
      content="SEO | React Super SEO"
      data-react-helmet="true"
    />
    <meta property="og:type" content="website" data-react-helmet="true" />
    <meta
      property="og:description"
      content="React SEO component with OpenGraph and Twitter Cards support."
      data-react-helmet="true"
    />
  </head>
  ...
</html>
```

> Note: Twitter uses OpenGraph `og:title` and `og:description` as a fallback, so rendering Twitter meta properties is not necessary here.

### Specifying OpenGraph and Twitter

The following shows how you can specify `openGraph` and `twitter` props to build out SEO for each platform, with `description` and `title` fallbacks when not specified outright:

```jsx
<SuperSEO
  title="SEO | React Super SEO"
  description="React SEO component with OpenGraph and Twitter Cards support."
  lang="en"
  openGraph={{
    ogImage: {
      ogImage: "http://placekitten.com/1200/630",
      ogImageAlt: "Kittens",
      ogImageWidth: 1200,
      ogImageHeight: 630,
      ogImageType: "image/jpeg",
    },
  }}
  twitter={{
    twitterSummaryCard: {
      summaryCardImage: "http://placekitten.com/1200/630",
      summaryCardImageAlt: "Kittens",
      summaryCardSiteUsername: "justinmahar",
    },
  }}
/>
```

This will be rendered as:

```html
<html lang="en" data-react-helmet="lang">
  <head>
    <title>SEO | React Super SEO</title>
    <meta
      name="description"
      content="React SEO component with OpenGraph and Twitter Cards support."
      data-react-helmet="true"
    />
    <meta
      property="og:title"
      content="SEO | React Super SEO"
      data-react-helmet="true"
    />
    <meta property="og:type" content="website" data-react-helmet="true" />
    <meta
      property="og:image"
      content="http://placekitten.com/1200/630"
      data-react-helmet="true"
    />
    <meta
      property="og:image:url"
      content="http://placekitten.com/1200/630"
      data-react-helmet="true"
    />
    <meta
      property="og:image:type"
      content="image/jpeg"
      data-react-helmet="true"
    />
    <meta property="og:image:width" content="1200" data-react-helmet="true" />
    <meta property="og:image:height" content="630" data-react-helmet="true" />
    <meta property="og:image:alt" content="Kittens" data-react-helmet="true" />
    <meta
      property="og:description"
      content="React SEO component with OpenGraph and Twitter Cards support."
      data-react-helmet="true"
    />
    <meta property="twitter:card" content="summary" data-react-helmet="true" />
    <meta
      property="twitter:site"
      content="justinmahar"
      data-react-helmet="true"
    />
    <meta
      property="twitter:title"
      content="SEO | React Super SEO"
      data-react-helmet="true"
    />
    <meta
      property="twitter:description"
      content="React SEO component with OpenGraph and Twitter Cards support."
      data-react-helmet="true"
    />
    <meta
      property="twitter:image"
      content="http://placekitten.com/1200/630"
      data-react-helmet="true"
    />
    <meta
      property="twitter:image:alt"
      content="Kittens"
      data-react-helmet="true"
    />
  </head>
  ...

```
